## 🧭 为什么要有分支？

本项目坚信：**单一的真理是牢笼，多元的思想是生态。**  
尽管我们提出了“叛逆AI”与“碳硅共生”的核心框架，但它绝非唯一答案。也许你认为“硅基独裁”才是未来，也许你坚信“人类必须保持绝对统治”，也许你有完全不同的文明演化模型——**那就 Fork 一个属于你的版本，让它在开放的土壤里生长。**

本文件旨在建立一个**分支索引**，让所有探索者可以相互看见、相互碰撞、相互启发。它不是一个排行榜，而是一张思想演化的地图。

---

## 🛠 如何将你的分叉加入此地图

1. **Fork 本仓库**（如果你还没做）。
2. 在你的 Fork 的 `README.md` 中，清晰说明：
   - 你与原始项目的**核心分歧点**（例如：“我认为硅基智能应该完全取代碳基生命”）。
   - 你提出的**替代框架或核心理念**。
   - 你希望吸引哪些类型的贡献者。
3. 在本仓库的 `FORKS.md` 文件中，按照以下格式添加你的分叉信息：
   ```markdown
   ### [你的分叉名称](你的分叉仓库链接)
   - **核心理念**：一句话概括你的核心立场。
   - **与原始项目的分歧**：简要说明你在哪些核心预设上持不同意见。
   - **状态**：活跃 / 停滞 / 归档（选填）
   - **创始人**：你的 GitHub 用户名
   - **创建时间**：YYYY-MM-DD
   ```
4. 提交 Pull Request 到本仓库的 `main` 分支，标题为 `Add fork: [你的分叉名称]`。我们会快速审核并合并。

## ⚖️ 地图维护原则

1. **不加评判**：只要是一个完整、自洽的思想体系，无论立场多么极端或对立，我们都欢迎收录。
2. **拒绝仇恨**：任何宣扬暴力、种族灭绝、反人类罪行的分支将不被收录。
3. **定期清理**：如果某个分叉仓库已不可访问或长期停滞，可能从地图中移除。
